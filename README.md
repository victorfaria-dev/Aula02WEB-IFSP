Atividade da segunda aula de Desenvolvimento Web da minha faculdade de Análise
e Desenvolvimento de Sistemas. A atividade tratava de fazer apenas alguns parágrafos,
utilizando tags basicas "h1", "p", "hr"... mas resolvi criar uma simples
interface de redirect para a atividade. O professor pode fácilmente copiar meu nome
e prontuário, acessar o a atividade para seu link direto, no qual está o comando.
Basicamente tudo foi montado por mim, a excessão do botão "Ir", utilizando os conceitos
básicos aprendidos préviamente em aula. Este foi o primeiro "projeto" meu, detalhado, em
HTML. Aprendi diversas funcionalidades do VSCode, novas funções, atalhos, lógicas... e 
durante o processo o básico de Git e GitHub.
