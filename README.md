Atividade da segunda aula de Desenvolvimento Web da minha faculdade de Análise e
Desenvolvimento de Sistemas. A atividade tratava de fazer apenas alguns
parágrafos, utilizando tags básicas como "h1", "p", "hr"... mas resolvi criar
uma simples interface de redirect para a atividade. O professor pode facilmente
copiar meu nome e prontuário, acessar a atividade por meio do link direto, no
qual está o comando. Basicamente, tudo foi montado por mim, à exceção do botão
"Ir", utilizando os conceitos básicos aprendidos previamente em aula. Este foi o
meu primeiro "projeto" mais detalhado em HTML. Aprendi diversas funcionalidades
do VSCode, novas funções, atalhos, lógicas... e, durante o processo, o básico de
Git, GitHub, CSS, JavaScript e como fazê-los funcionar juntos.
